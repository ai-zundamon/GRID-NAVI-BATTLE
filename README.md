# GRID NAVI BATTLE

6×3 のグリッド上でリアルタイムに戦う、オリジナルのアクションバトルゲームです。単一の HTML ファイルのみで動作し、外部ライブラリやビルド工程は不要です（Vanilla JavaScript + HTML Canvas + Web Audio）。通信は一切行わず、すべてブラウザ内で完結します。

## 遊び方

ブラウザで `index.html` を開くだけでプレイできます。GitHub Pages で公開している場合は、その公開 URL にアクセスしてください。PC・スマートフォンのどちらにも対応しています。

## ゲーム内容

- 6×3 グリッドでのリアルタイム戦闘
- コードを揃えて複数同時にセットするチップ選択（カスタム画面）
- 特定のチップの組み合わせで発動する強力な連携技（P.A.）
- 難易度 EASY / NORMAL / HARD、および解放後に挑める EX モード
- 多段階で変化するボス戦
- 実績（メダル）システムと、達成すると解放される報酬チップ
- チップ図鑑（基本 12 種＋報酬 5 種＝全 17 種）
- バスターのみでも全難易度クリア可能なバランス設計

## 操作方法

### スマートフォン（タッチ）

- 移動：画面下部の十字ボタン（D-pad）
- ショット：`B` ボタン
- チップ使用：`CHIP` ボタン
- カスタム画面を開く：`CUST` ボタン
- タイトル画面：D-pad で「スタート／図鑑／実績」を選び `B` で決定（各ボタンを直接タップしても選べます）
- どの画面からでもタイトルへ戻る：画面上部の「タイトル画面へ戻る」ボタン
- この説明（README）を開く：画面上部の `README` ボタン

### PC（キーボード／マウス）

- 移動：方向キー または `W` `A` `S` `D`
- ショット：`Z`
- チップ使用：`X`
- カスタム画面を開く：`C`
- タイトル画面：方向キーで「スタート／図鑑／実績」を選び `Z` または `Enter` で決定（`X`＝図鑑、`C`＝実績 のショートカットも利用可）
- 設定：`M`＝BGM、`F`＝画面エフェクト、`V`＝画面シェイク
- どの画面からでもタイトルへ戻る：`Esc` キー、またはマウスの右クリック
