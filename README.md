# GRID NAVI BATTLE

グリッド戦闘ゲームです。単一の HTML ファイルのみで動作し、外部ライブラリやビルド工程は不要です（Vanilla JavaScript + HTML Canvas）。

## 遊び方

ブラウザで `index.html` を開くだけでプレイできます。GitHub Pages で公開した場合は、公開 URL にアクセスしてください。

## 操作方法

**PC（キーボード）**

- 移動：方向キー または `W` `A` `S` `D`
- ショット：`Z`
- チップ使用：`X`
- カスタム画面：`C`
- 図鑑：`X`（タイトル画面）／ 実績：`C`（タイトル画面）

**スマートフォン**

- 画面下部の十字ボタンで移動、`CUST`／`CHIP`／`B` ボタンで操作

## 備考

- セーブデータはお使いのブラウザの localStorage に保存されます。
- 通信は行わず、すべてブラウザ内で完結します。
